Move Size:

Setup the Volume/Weight of each Move Size description. It populates into the Select Move Size dropdown selection when creating a New Estimate.
You can inactivate a Move Size description or add one.

The target is to generate a quick, ballpark estimate based on the size of move.

As customary in any type of move, ask the customer for a detailed inventory list or send the customer an email to fill it out on his own.
