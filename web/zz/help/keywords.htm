Keywords Definition
Add Keywords in your email content. Each Keyword converts into the customer's data.

JOBNO: Estimate/Job Number
CLIENTNAME: Customer First Name (Moving From)
CLIENTTO: Customer First Name (Moving To)
MOVEDAY: Day of the Move
MOVEDATE: 1st Pick-Up Date
MOVETIME: Pick-Up Time
FROMADD: Origin Address (address, city, state, zip code)
TOADD: Destination Address (address, city, state, zip code)
FROMSTATE: Origin State Only
TOSTATE: Destination State Only
FROMCOUNTRY: Origin Country (International)
TOCOUNTRY: Destination Country (International)
TOTALJOB Total Estimate
ESTUSER: User assigned to the Estimate
USER-PHONE: User Direct Phone Number
PHONE1: First Phone Number (From Moving From)
PHONE2: Second Phone Number (From Moving From)
PHONE3: Second Phone Number (From Moving To)
MOBLIE: Cell Phone (From Moving To)
FOLLOW-UP: Follow-Up Date
FOLLOW-TIME: Follow-Up Time
LOCAL-VAN: Number of Van on Local Move
LOCAL-MEN: Number of Workers on Local Move
LOCAL-PERHRS: Rate per Hour on Local Move
RATE-PER: Rate per CF or LBS
TOTALCF: Estimated CF
TOTALLBS: Estimated LBS
ESTIMATOR: Onsite Visual Estimator
VISUALDAY: Day of Visual Appointment
VISUALDATE: Onsite Visual Date/Appointment
VISUALTIME: Onsite Visual Appointment Time
DLVDATEA: 1st Requested Delivery Date (by customer)
DLVDATEB: 2nd Requested Delivery Date (by customer)
PACKDATE: Packing Date
PACKTIME: Packing Time
PAYMENTS: Total Payments
