Users Table

Create a list of office employees that will be utilizing the Moving Software from position of Sales, Dispatch to Office/Sales Management and Administration.
Each User profile comes with authorization (what the User CAN NOT DO) and privileges (what the User CAN DO). A User can serve several Departments (With the Department feature).

User Authorization:
Block a User from operating certain functions based on the list provided. Blocking the User means the User can't utilize said option.

User Privileges:
Allow a User to operating certain functions by adding the relevant privilege.

Commission: (Advance Package).
Pay weekly salary and commission for booking jobs. A User can get commission either from the estimate or actual total of the job. To select this option use the Estimate Commission checkbox.

Security: User can only work from a specific IP.
Limit a User to work from one location (office, home) by indicating the IP given by the Internet Service Provider (ISP). Utilize this option only if your ISP provides your/user location with a Static IP.

Inactivate or delete a User in case he/she no longer works in the company. It's recommended to transfer the User's estimates to another user.

Click on "Add User" from the Users Table and fill out the User's information.
Click submit to save any changes that you made.
