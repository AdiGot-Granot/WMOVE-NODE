Setup Prices:

Review and select the relevant options for your company in order to comply with your way of work.

Section 1: Basic Setup

Calculate long distance rates by Cu. Ft.: Selected by default. Uncheck to switch to Lbs.

Show Tariff on Printed Estimate and Charges Form: Show the Tariff on the Moving Estimate and Charges screen.

Show Packing Materials List on Printed Estimate: Select to show your Packing Materials list and pricing on the Moving Estimate.

Show Only Selected Packing Materials on Printed Estimate: Select to show ONLY the Packing Materials you/the customer selected on the Moving Estimate.

Show Flat Rate on Printed Estimate for Local Move: Select if you DON'T want to show the local labor and travel time pricing breakdown.

Block Advertising Source/List on Online Estimate Link/Moving Estimate: Removes the Lead source name and list of Lead Providers.

Customer Update Inventory Link (Skip entry form): Sends the customer directly to the inventory screen.

Allow Duplicate Leads from Website: Allow customers to submit more then one estimate with the same email. Applies only when the Free Estimate form on the website is connected to the Moving Software.

Set Default Binding Type: 0=None, 1=Binding, 2=Non-Binding. Long distance estimates only.

Customer Inventory Form: Setup which inventory form customers will use to update their inventory (0=Rooms, 1=ABC).

Add Sales Tax on Packing Materials: Some states requires charging sales tax on packing materials. Select if applies.

Senior Citizen/Coupon Discount: Set default discount percentage.

Click submit to save any changes that you made.
