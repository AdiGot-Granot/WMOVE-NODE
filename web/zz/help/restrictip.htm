Restricted IP

Block a single IP or IP series from submitting fake estimates. This option is relevant when the company website is integrated with the Moving Software via a Free Quote/Estimate form and when you run an online PPC (Pay Per Click) campaign.

Use the Client/Users Activities report to identify a client IP that constantly creates fake web estimates.

Important

This option is not meant to restrict Users from logging in to the Moving Software. Do not restrict yourself!!! Your IP is presented to you as "My IP".

Click submit to save any changes made.
