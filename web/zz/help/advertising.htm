Advertising Table

Create a list of advertising sources that provide you with moving leads. Advertising source appears under How did you hear about us? when entering New Estimates by data entry.

Indicating the advertising source is mandatory when creating a new estimate. Users will be asked to select which advertising source guided the customer to become familiar with the moving company.

A basic list of advertising sources is provided to you. You can add, update or delete any advertising source if applicable.

You also get a Leads & Advertising report that produces leads statistics. This will help you to determine which advertising source has quality leads based on booking ratio.

Please note:
1. When receiving moving leads automatically, the Lead Provider name will automatically appear on the estimate.
2. You can block showing the advertising source on the printed estimate form. Please refer to the Setup Prices section.

Click submit to save any changes that you made.
